# GET /cuentas/hola
curl -i -X GET "http://localhost:8082/cuentas/hola"

# GET /cuentas (lista)
curl -i -X GET "http://localhost:8082/cuentas" -H "Accept: application/json"

# GET /cuentas/{id}
curl -i -X GET "http://localhost:8082/cuentas/12345" -H "Accept: application/json"

# POST /cuentas (crear)
curl -i -X POST "http://localhost:8082/cuentas" \
  -H "Content-Type: application/json" \
  -d '{
    "nroCuenta":"12345",
    "usuario":"juan",
    "monto":1000.00,
    "estado":"ACTIVA",
    "tipoCuenta":"CAJA_AHORRO"
  }'

# PUT /cuentas/{id} (actualizar)
curl -i -X PUT "http://localhost:8082/cuentas/12345" \
  -H "Content-Type: application/json" \
  -d '{
    "nroCuenta":"12345",
    "usuario":"juan_actualizado",
    "monto":1500.00,
    "estado":"ACTIVA",
    "tipoCuenta":"CAJA_AHORRO"
  }'

# DELETE /cuentas/{id}
curl -i -X DELETE "http://localhost:8082/cuentas/12345"


//asoxiar o desasociar usuarios a una cuenta
Este endpoint añade el ID de un usuario a la lista de usuarios asociados a una cuenta específica.
Método HTTP: PUT
URL: /cuentas/{nroCuenta}/asociar-usuario/{idUsuario}
Descripción:
{nroCuenta}: Es el número de la cuenta que quieres modificar.
{idUsuario}: Es el ID del usuario que quieres asociar a esa cuenta.

Ejemplo de uso con Postman o cURL:
PUT http://localhost:8082/cuentas/1/asociar-usuario/1

2. Desasociar un Usuario de una Cuenta
Este endpoint elimina el ID de un usuario de la lista de usuarios asociados a una cuenta.
Método HTTP: PUT
URL: /cuentas/{nroCuenta}/desasociar-usuario/{idUsuario}
Descripción:
{nroCuenta}: Es el número de la cuenta que quieres modificar.
{idUsuario}: Es el ID del usuario que quieres quitar de la cuenta.
Ejemplo de uso con Postman o cURL:

PUT http://localhost:8082/cuentas/C-1001/desasociar-usuario/1
